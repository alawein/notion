# AI Ecosystem Hierarchy

🎯 Artificial Intelligence (AI) ROOT Broadest umbrella · all intelligent machine systems All technologies aimed at creating machines that perform tasks requiring human-like intelligence. Ranges from simple IF-THEN rules to modern neural networks. Rule-based systems Expert systems Decision trees Machine Learning Deep Learning ▾ Learning Paradigms 📈 Machine Learning (ML) CORE SUBSET Algorithms that learn patterns from data · no explicit programming Enables machines to learn from data and improve performance without being explicitly programmed. Subdivides into Supervised, Self-Supervised, Reinforcement, and Deep Learning. Supervised Self-Supervised Reinforcement Deep Learning ▾ 🏷️ Supervised Learning (SL) Labeled data (X→Y) · classification & regression Train on labeled examples where correct outputs are known. Two major tasks: Classification (predict categories) and Regression (predict numbers). Spam detection Loan approval House prices Fraud detection ▾ Classification · predict categories Binary (yes/no) or multi-class (A/B/C). Output is a discrete label. Email: SPAM / NOT SPAM Loan: APPROVE / DENY Disease: YES / NO ▾ Regression · predict numbers Output is a continuous value. Minimize prediction error (MSE/MAE). House price: $450K Sales forecast: $2.5M Stock price: $148 ▾ 🔄 Self-Supervised Learning (SSL) Model generates its own labels · unlabeled data A nuanced form of unsupervised learning. The model creates supervision signal from input (e.g., predict masked words). Key for pre-training foundation models. BERT: masked word prediction GPT: next-token prediction Contrastive learning ▾ 🎮 Reinforcement Learning (RL) Agent · Environment · Reward signal · No labeled data An agent learns by interacting with an environment. Actions producing good outcomes get positive rewards; bad ones get penalized. Maximizes cumulative reward. AlphaGo Robotic control Game AI RLHF for LLMs ▾ Neural Architectures 🧬 Deep Learning (DL) SUBSET OF ML Multi-layer neural networks · automatic feature learning Uses neural networks with many stacked layers to learn hierarchical representations directly from raw data. Drives breakthroughs in vision, speech, and language. ANN CNN RNN LSTM Transformer ▾ 1️⃣ ANN — Artificial Neural Network Input → Hidden layers → Output · tabular/structured data The foundational architecture. Each neuron computes a weighted sum of inputs + bias, passes through an activation function. Training: forward pass → loss → backprop → gradient descent. Loan approval Sales forecasting Churn prediction Priority classification ▾ 2️⃣ CNN — Convolutional Neural Network Convolution → Max Pool → Flatten → ANN · images & spatial data Filters scan the image extracting edges, textures, and shapes. Max pooling compresses feature maps. Flattening converts to a vector fed into a dense ANN for classification. Face recognition Tumor detection Self-driving cars iPhone FaceID ▾ Steps: Convolution → Pooling → Flatten → Classify Convolution: 3×3 filter scans image → Feature map. 
Max Pooling: 32×32 → 16×16 (keep strongest). 
Flatten: 2D → 1D vector [2,8,5,3…] (4096 values). 
Dense ANN: [Dog 92%, Cat 5%, Bird 3%] ▾ 3️⃣ RNN — Recurrent Neural Network Hidden state loops back · short-term memory · sequences Previous output (hidden state H_t-1) becomes input at next step. Remembers context within a sequence. Suffers vanishing gradient over long distances (~7–10 steps max). Stock prediction Sentiment analysis Autocomplete Speech-to-text ▾ Seq2Seq variants One→One: image class. 
One→Many: image → caption. 
Many→One: reviews → sentiment. 
Many→Many: translation ("Hello" → "Bonjour"). ▾ 4️⃣ LSTM — Long Short-Term Memory RNN upgrade Forget · Input · Output gates · 100+ step memory Solves vanishing gradient via gated memory cells. Forget gate discards old info; Input gate adds new; Output gate filters what to send forward. Memory cell flows unchanged through time. Language translation Chatbots Long time-series Subtitle generation ▾ 5️⃣ Transformer powers LLMs Self-attention · parallel processing · no recurrence needed Replaces recurrence with self-attention: every token attends to every other token simultaneously. Massively parallelizable; scales to billions of parameters. Foundation of all modern LLMs. BERT GPT family T5 Vision Transformer ▾ Generative & Foundational ✨ Generative AI CREATES new data GANs · Diffusion · Foundation models · LLMs Models that generate new content (images, text, audio, video) resembling training data. Overlaps with Deep Learning at the architecture level. DALL-E Stable Diffusion Midjourney GPT-4 Claude ▾ 🎨 GANs — Generative Adversarial Networks Generator vs Discriminator · competitive training loop Two networks compete: Generator creates fakes from noise; Discriminator judges real vs fake. Competition drives photorealistic generation. Synthetic faces Art generation Deepfakes Image super-resolution ▾ 🌊 Diffusion Models Noise → iterative denoising → coherent image Forward process: gradually add noise to images. Reverse process: learn to remove noise step by step. Conditioned on text prompts for text-to-image. DALL-E 3 Stable Diffusion Midjourney Imagen ▾ 🏗️ Foundational Models pre-train → fine-tune Large pre-trained on massive data · versatile base Trained via self-supervised learning on enormous corpora. Capture broad world knowledge. Fine-tuned (SFT) for specific downstream tasks with small labeled datasets. BERT GPT-3 T5 PaLM LLaMA ▾ 🗣️ Large Language Models (LLMs) LEAF Transformer-based · billions of params · human-like text Specialized foundational models for language tasks. Training pipeline: Pre-training (SSL) → SFT (Supervised) → RLHF (RL). Capable of reasoning, coding, translation, and creative writing. ChatGPT Claude Gemini GPT-4 LLaMA 3 ▾ SFT — Supervised Fine-Tuning Supervised Learning Labeled (prompt, ideal-response) pairs. Teaches the base model to follow instructions. Falls under Supervised Learning paradigm. Q&A examples Instruction following ▾ RLHF — Reinforcement Learning from Human Feedback Reinforcement Learning Human raters rank model responses. A reward model is trained on these preferences, then used to fine-tune the LLM via PPO. Falls under RL paradigm. Helpfulness alignment Safety training ▾
